Build a recipe blog for Frida Sofia Eats
Business Background:
    1. Frida Sofia Eats is an Instagram creator (https://www.instagram.com/fridasofiaeats/) that posts recipe videos. 
    2. The IG page links to recipes on this blog (current version is running on Squarespace but we are building its replacement. link is: fridasofiaeats.com)

Business Goals: 
    1. Build an engaging site that will entice users to explore more pages (maximizing ad revenue)
    2. Build up a platform for various monetization opportunities (paid partnerships, products, referrals etc.)
    3. Frida plans to expand to other platforms and having the website as a place to launch that will be helpful  

Site Description:
    1. Header - Include the company logo and make a compelling header where users can navigate across the site
    2. Home section - 
        - Here we will design page that entices users to check out different recipes on the site. 
        - We can also look at adding fun interactive features related to 
    3. Footer - Needs a footer for privacy statement and Raptive ad legal lingo
    4. Blog - There needs to be the ability to see a collection of posts on one page. they can be grouped by different categories (With overlap). 
    5. Blog post
        - Each blog post should have room for the following sections:
            - user actions: 
                - users should be able to print the recipe (format the text so that only the recipe shows up).
                - Users should be able to copy the recipe to their clipboard (again it should be formatted down to just the text)
                - Users should be linked to videos of the recipe on other platforms
            - Intro text section
                - plain text
            - Ingredients section 
                - plain text 
            - Instructions section
                - plain text
            - Any additional sections that would help further the business goals
            - Section for beautiful images of the dish
    6. Blog post creator 
        - Provide functionality to create new blog posts
        - This should also include looking into automation with instagram where a blog post is published when the instagram post is put up
        - This can be established via webhook

Backend Needs:
    1. System to support blog
    2. Look to the future in case we want to have users who can sign up for a newsletter
    3. Handle the ability for users to review recipes without logging in.
